1、定义一个数组HEROES

const HEROES: Hero[] = []

2、在appcomponent上定义一个全局属性，暴漏上面定义的数组

heroes = HEROES;

3、使用ngFor来循环显示列表

<li *ngFor="let hero of heroes" (click)="onSelect(hero)">
<span class="badge">{{hero.id}}</span> {{hero.name}}
</li>

4、插入列表单击事件

5、在app.component.ts中添加selectHero属性，用以绑定选中列表数据

selectedHero: Hero;
onSelect(hero: Hero): void {
this.selectedHero = hero;
}

6、添加ngIf，处理初始化时没有选中的情况

7、为所选li行添加样式(当双引号的值为true的时候，会添加selected样式，否则移除)

<li *ngFor="let hero of heroes"
[class.selected]="hero === selectedHero"
(click)="onSelect(hero)">
