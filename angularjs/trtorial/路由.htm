路由是导航的另一个名字。路由器就是从一个视图导航到另一个视图的机制。

1、查分主视图，使主视图只处理路由导航

将主视图分离处理，组建列表单独放到一个组件中，将数据服务包喊道ngModule中，这样服务不用在自组件提供商数组中单独声明了

2、配置录用

a、添加base组件：<base href="/">。这是必须的

b、在ngModule中添加路由信息

  1. import { RouterModule } from '@angular/router';
  1. imports: [
  2. BrowserModule,
  3. FormsModule,
  4. RouterModule.forRoot([
  5. {
  6. path: 'heroes',
  7. component: HeroesComponent
  8. }
  9. ])
  10. ],

c、添加出口outlet标签和超链接

template: ` <h1>{{title}}</h1> <a routerLink="/heroes">Heroes</a> <router-outlet></router-outlet> `

3、配置仪表盘路由

a、添加仪表盘组件文件；b、在ngModule中添加组引用并且添加路由跳转配置

4、路由列表详情

a、添加占位符路由配置

{ path: 'detail/:id', component: HeroDetailComponent },

b、修改组件信息

ngOnInit(): void {
this.route.params
.switchMap((params: Params) => this.heroService.getHero(parseInt(params['id'], 10)))//+params['id']转数字
.subscribe(hero => this.hero = hero);
};
