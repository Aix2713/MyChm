对表格或者表单数据进行验证，验证的是model中定义的字段验证

var me = this,
view = this.getView().down('grid'),
gStore = me.getViewModel().get('gridStore'),
gData = gStore.getData();
for (var i = 0; i < gData.getCount(); i++) {
if (!gData.getAt(i).isValid()) {
console.log(1)
}
}
console.log(gStore.getData().getAt(0).validate());

note:不能对所有数据进行一次性验证，只能循环对单独数据进行验证！
