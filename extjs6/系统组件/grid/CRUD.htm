增加(Create)、查询(Retrieve)（重新得到数据）、更新(Update)和删除(Delete)

1、新增操作

view = me.getView().down('grid'),
gStore = me.getViewModel().get('TagsStore');
gStore.insert(gStore.getCount(), {
code: '',
icon: '',
name: ''
});
view.plugins[0].startEditByPosition({
row: gStore.getCount(),
column: 0
});

2、删除

gStore.remove(view.getSelection()[0]);

gStore表示当前view的store，view表示当前grid
